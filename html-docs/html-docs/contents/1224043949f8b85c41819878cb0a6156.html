Visualizar Ordem de Compra

UMLUseCase
Untitled :: Casos de Uso :: Visualizar Ordem de Compra

Description

Fluxo Principal:

O usuário acessa a opção de visualização de ordens de compra. O sistema exibe uma lista das ordens de compra do usuário. O usuário seleciona uma ordem para obter detalhes adicionais. O sistema apresenta os detalhes completos da ordem selecionada.

Fluxo Alternativo - Nenhuma Ordem Encontrada:

No passo 2, se não houver ordens de compra: O sistema informa ao usuário que não há ordens encontradas. O usuário pode optar por realizar uma nova ordem de compra.
