Modificar Ordem de Compra

UMLUseCase
Untitled :: Casos de Uso :: Modificar Ordem de Compra

Description

Fluxo Principal:

O usuário acessa a opção de modificação de ordens de compra. O sistema exibe uma lista das ordens de compra do usuário. O usuário seleciona a ordem que deseja modificar. O sistema exibe os detalhes da ordem e permite que o usuário faça modificações (por exemplo, altera a quantidade). O sistema valida as modificações. Se as modificações são válidas, o sistema atualiza a ordem de compra. O sistema emite uma confirmação de modificação bem-sucedida.

Fluxo Alternativo - Modificações Inválidas:

No passo 5, se as modificações são inválidas: O sistema notifica o usuário sobre as modificações incorretas. O usuário corrige as informações e tenta novamente.
