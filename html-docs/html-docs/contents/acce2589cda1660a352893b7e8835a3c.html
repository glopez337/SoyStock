Visualizar carteira de ações

UMLUseCase
Untitled :: Casos de Uso :: Visualizar carteira de ações

Description

Fluxo Principal:

O usuário acessa a opção de visualização da carteira de ações. O sistema exibe uma lista das ações presentes na carteira do usuário, juntamente com suas quantidades. O usuário pode filtrar ou ordenar a lista conforme necessário. O sistema apresenta informações adicionais sobre cada ação, como valor atual, variação, etc. O usuário pode navegar para obter detalhes mais específicos sobre cada ação.

Fluxo Alternativo - Carteira Vazia:

No passo 2, se a carteira do usuário estiver vazia: O sistema informa ao usuário que a carteira está vazia. O usuário pode optar por realizar uma compra de ações
