Enviar com Gatilho

UMLUseCase
Untitled :: Casos de Uso :: Enviar com Gatilho

Description

Fluxo Principal:

O usuário acessa a opção de envio de ordens de venda com gatilho. O sistema exibe a lista das ações presentes na carteira do usuário, juntamente com suas quantidades disponíveis para venda. O usuário seleciona a ação que deseja vender. O sistema solicita a quantidade de ações que o usuário deseja vender e a configuração do gatilho. O usuário insere a quantidade desejada e configura o gatilho (por exemplo, preço de disparo). O sistema verifica a disponibilidade da quantidade de ações para venda. Quando o preço atinge o gatilho configurado, o sistema automaticamente envia a ordem de venda. O sistema valida a ordem e processa a transação. O sistema atualiza a carteira do usuário e registra a ordem no histórico. O sistema emite uma confirmação de venda bem-sucedida.

Fluxo Alternativo - Gatilho não Alcançado:

No passo 7, se o preço não atingir o gatilho configurado: O sistema aguarda até que o preço alcance o gatilho. Quando o gatilho é alcançado, o sistema continua com o fluxo principal.
