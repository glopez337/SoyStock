Modificar ações

UMLUseCase
Untitled :: Casos de Uso :: Modificar ações

Description

Fluxo Principal:

O usuário acessa a opção de modificação de ações na carteira. O sistema exibe uma lista das ações presentes na carteira, juntamente com suas quantidades. O usuário seleciona a ação que deseja modificar. O sistema exibe opções para modificar a quantidade de ações (compra ou venda). O usuário insere a quantidade desejada. O sistema verifica a disponibilidade e valida a modificação. Se a modificação é válida, o sistema atualiza a carteira do usuário. O sistema emite uma confirmação de modificação bem-sucedida.

Fluxo Alternativo - Quantidade Indisponível para Venda:

No passo 6, se a quantidade desejada para venda não estiver disponível: O sistema informa ao usuário sobre a quantidade indisponível. O usuário pode optar por modificar a quantidade ou cancelar a modificação.

Fluxo Alternativo - Quantidade Indisponível para Compra:

No passo 6, se a quantidade desejada para compra não estiver disponível: O sistema informa ao usuário sobre a quantidade indisponível. O usuário pode optar por modificar a quantidade ou cancelar a modificação.
