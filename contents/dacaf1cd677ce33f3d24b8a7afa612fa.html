Enviar Ordem de Venda

UMLUseCase
Untitled :: Casos de Uso :: Enviar Ordem de Venda

Description

Fluxo Principal:

O usuário acessa a opção de envio de ordens de venda. O sistema exibe a lista das ações presentes na carteira do usuário, juntamente com suas quantidades disponíveis para venda. O usuário seleciona a ação que deseja vender. O sistema solicita a quantidade de ações que o usuário deseja vender. O usuário insere a quantidade desejada. O sistema verifica a disponibilidade da quantidade de ações para venda. O usuário confirma a ordem de venda. O sistema valida a ordem e processa a transação. O sistema atualiza a carteira do usuário e registra a ordem no histórico. O sistema emite uma confirmação de venda bem-sucedida.

Fluxo Alternativo - Quantidade Indisponível:

No passo 6, se a quantidade desejada para venda não estiver disponível: O sistema informa ao usuário sobre a quantidade indisponível. O usuário pode optar por modificar a quantidade ou cancelar a ordem.
