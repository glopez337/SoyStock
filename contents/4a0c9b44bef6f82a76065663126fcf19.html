Visualizar Ordem de Venda

UMLUseCase
Untitled :: Casos de Uso :: Visualizar Ordem de Venda

Description

Fluxo Principal:

O usuário acessa a opção de visualização de ordens de venda. O sistema exibe uma lista das ordens de venda do usuário. O usuário seleciona uma ordem para obter detalhes adicionais. O sistema apresenta os detalhes completos da ordem de venda selecionada.

Fluxo Alternativo - Nenhuma Ordem Encontrada:

No passo 2, se não houver ordens de venda em andamento: O sistema informa ao usuário que não há ordens encontradas. O usuário pode optar por realizar uma nova ordem de venda.
