Modificar Ordem de Venda

UMLUseCase
Untitled :: Casos de Uso :: Modificar Ordem de Venda

Description

Fluxo Principal:

O usuário acessa a opção de modificação de ordens de venda. O sistema exibe uma lista das ordens de venda em andamento do usuário. O usuário seleciona a ordem de venda que deseja modificar. O sistema exibe os detalhes da ordem de venda e permite que o usuário faça modificações (por exemplo, altera a quantidade). O usuário insere a quantidade desejada. O sistema verifica a disponibilidade e valida a modificação. Se a modificação é válida, o sistema atualiza a ordem de venda. O sistema emite uma confirmação de modificação bem-sucedida.

Fluxo Alternativo - Quantidade Indisponível para Modificação:

No passo 6, se a quantidade desejada para modificação não estiver disponível: O sistema informa ao usuário sobre a quantidade indisponível. O usuário pode optar por modificar a quantidade ou cancelar a modificação.
