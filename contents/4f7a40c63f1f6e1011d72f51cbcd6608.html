Modificar Cadastro

UMLUseCase
Untitled :: Casos de Uso :: Modificar Cadastro

Description

Fluxo Principal:

O usuário acessa a opção de modificação de cadastro. O sistema exibe o formulário atual com as informações cadastradas. O usuário faz as modificações desejadas (por exemplo, altera o endereço). O sistema valida as modificações. Se as modificações forem válidas, o sistema atualiza as informações do usuário. O sistema emite uma confirmação de modificação bem-sucedida.

Fluxo Alternativo - Informações Inválidas:

No passo 4, se as modificações são inválidas: O sistema notifica o usuário sobre as informações incorretas. O usuário corrige as informações e tenta novamente.
