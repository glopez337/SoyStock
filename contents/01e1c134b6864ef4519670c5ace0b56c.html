Enviar Ordem de Compra

UMLUseCase
Untitled :: Casos de Uso :: Enviar Ordem de Compra

Description

Fluxo Principal:

O usuário acessa a opção de envio de ordens de compra. O sistema exibe opções para seleção de ações e quantidade desejada. O usuário seleciona a ação e insere a quantidade desejada. O sistema verifica a disponibilidade da ação e calcula o valor total da transação. O usuário confirma a ordem de compra. O sistema valida a ordem e processa a transação. O sistema atualiza a carteira do usuário e registra a ordem no histórico. O sistema emite uma confirmação de compra bem-sucedida.

Fluxo Alternativo - Quantidade Indisponível:

No passo 4, se a quantidade desejada não estiver disponível: O sistema informa ao usuário sobre a quantidade indisponível. O usuário pode optar por modificar a quantidade ou cancelar a ordem.
