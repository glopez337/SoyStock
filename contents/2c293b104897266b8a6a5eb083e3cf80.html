Comprar Ações

UMLUseCase
Untitled :: Casos de Uso :: Comprar Ações

Description

Fluxo Principal:

O usuário inicia o processo de compra. O sistema exibe opções de ações disponíveis. O usuário seleciona a ação desejada e a quantidade. O sistema verifica a disponibilidade e calcula o valor total. O usuário confirma a compra. O sistema verifica o saldo do usuário. A transação é processada, a carteira é atualizada e a compra é registrada no histórico. O sistema emite um alerta confirmando a conclusão da compra.

Fluxo Alternativo - Saldo Insuficiente:

No passo 6, se o saldo do usuário for insuficiente: O sistema notifica o usuário sobre o saldo insuficiente. O usuário pode optar por modificar a quantidade de ações ou cancelar a compra.
