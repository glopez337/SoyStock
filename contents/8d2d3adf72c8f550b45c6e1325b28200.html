Autentinticar Usuário

UMLUseCase
Untitled :: Casos de Uso :: Autentinticar Usuário

Description

luxo Principal:

O usuário acessa a opção de cadastro. O sistema exibe o formulário de cadastro. O usuário preenche as informações necessárias (nome, e-mail, senha, etc.). O sistema valida as informações e cria a conta do usuário. O sistema emite uma confirmação de cadastro bem-sucedido.

Fluxo Alternativo - Informações Inválidas:

No passo 4, se as informações fornecidas forem inválidas: O sistema informa ao usuário sobre os campos incorretos. O usuário corrige as informações e tenta novamente.
