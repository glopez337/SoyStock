Simulação de Negócios

UMLUseCase
Untitled :: Casos de Uso :: Simulação de Negócios

Description

Fluxo Principal:

O usuário acessa a opção de simulação de negócios. O sistema exibe uma interface para simulação, incluindo opções para escolher ações, quantidade, e realizar a simulação. O usuário seleciona as ações desejadas e insere a quantidade que gostaria de simular. O sistema realiza a simulação com base nas condições de mercado atuais. O sistema exibe os resultados da simulação, incluindo o desempenho das ações escolhidas. O usuário pode analisar os resultados e fazer ajustes na simulação conforme necessário.

Fluxo Alternativo - Ajustes na Simulação:

No passo 6, se o usuário deseja ajustar a simulação: O sistema permite que o usuário faça modificações na escolha de ações, quantidade, ou outros parâmetros. O sistema recalcula os resultados da simulação com base nos ajustes realizados.

Fluxo Alternativo - Exibição de Dados Detalhados:

No passo 5, se o usuário deseja dados mais detalhados sobre o desempenho das ações: O sistema fornece informações detalhadas, como variação percentual, valores máximos e mínimos, etc. O usuário pode explorar dados adicionais para uma análise mais profunda.
